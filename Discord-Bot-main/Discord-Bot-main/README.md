## 🚀 Setup

### 1) Clone
```bash
git clone https://github.com/SazzzNiziyan/Discord-Bot.git
cd Discord-Bot
```

### 2) Install dependencies
Choose what matches your project:

**Node.js**
```bash
npm install
```

**Python**
```bash
pip install -r requirements.txt
```

### 3) Add environment variables
Create a `.env` file:

```env
DISCORD_TOKEN=YOUR_DISCORD_BOT_TOKEN
```

### 4) Run
**Node.js**
```bash
npm run start
```

**Python**
```bash
python main.py
```
